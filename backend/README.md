# 🚀 Fullstack Project

> A modern fullstack web application with CI/CD pipeline and code quality analysis.

---

## 📌 Overview

Ce projet est une application web complète conçue pour gérer efficacement les données via une API REST et une interface utilisateur moderne.

Il inclut l'authentification, la gestion de bases de données et des contrôles automatisés de la qualité du code grâce à SonarCloud.

Le projet repose sur une architecture modulaire séparant le backend et le frontend pour une meilleure scalabilité et une maintenabilité accrue.

---

## ✨ Features

- 📊 Endpoint utilisé : api/lead_score
- 🔄 CI/CD pipeline with GitHub Actions
- 📈 Analyse de code avec SonarCloud

---

## 🧱 Tech Stack

### Backend

- Node.js
- Express.js
- Testeur de requête: Insomnia, Thunder Client, curl,....

### DevOps / Tools

- GitHub
- GitHub Actions
- SonarCloud

## ⚙️ Installation

### 1. Clone repository

```bash
git clone https://github.com/valisoa2002/SmartCRM.git
cd SmartCRM
```

### 2. Installation des dépendances de backend

```bash
cd backend
npm install
```

### 3. Création de fichier .env

```bash
touch .env
```

Ajoutez vos variables d'environnement (ex: PORT=3000)

### 4. Lancement du serveur

```bash
npm start
```
